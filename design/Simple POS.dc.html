Product Design Package
KRS — Simple POS ที่เชื่อมข้อมูลกับฐานข้อมูล KRS
ระบบขายหน้าร้านแบบ web/tablet สำหรับร้านเล็ก–กลาง เน้นขายเร็ว ปิดรอบง่าย และเตรียมข้อมูลให้พร้อมส่งเข้าระบบบัญชีภายนอก (PEAK, FlowAccount, Xero, QuickBooks, Custom API) โดยไม่ทำให้รกแบบ ERP
ขายเร็วบนแท็บเล็ต
แตะสินค้า → ตะกร้า → รับเงิน จบใน 3 ขั้น ปุ่มใหญ่ แตะง่าย เหมาะกับ cashier มือใหม่
Sync queue กันล่ม
ทุกบิลเข้า queue ขายต่อได้แม้บัญชีล่ม retry อัตโนมัติ มองเห็นสถานะชัด
แยกเลข POS / บัญชี
เลขบิล POS คนละชุดกับเลขเอกสารบัญชี daily summary เป็นค่าเริ่มต้น
หลักการออกแบบ · Design principles
01
ขายมาก่อน บัญชีตามหลัง
POS ขายได้เสมอ แม้ระบบบัญชีล่ม งานบัญชีเป็น async
02
ห้ามลบบิล
ใช้ cancel / void / refund เท่านั้น เพื่อรอยตรวจสอบครบ
03
สถานะบัญชีต้องมองเห็น
Unsynced / Synced / Failed / Retry มีสีและ badge ชัด
04
error ต้อง actionable
บอกสาเหตุเป็นภาษาคน + ปุ่มแก้ไข/retry ทันที
05
MVP ไม่ใช่ ERP
ตัดความซับซ้อน เหลือสิ่งที่ร้านต้องใช้จริง
06
ต่อยอด multi-branch ได้
MVP สาขาเดียว แต่โครงข้อมูลเผื่อหลายสาขา
